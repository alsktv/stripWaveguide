📚 Silicon Photonics Waveguide Real-time Simulator Docs이 문서는 실리콘 포토닉스(Silicon Photonics) C-band($1550\text{ nm}$) 도파로의 전자기장(EM Field) 분포, 전반사(TIR), 에바네센트 파(Evanescent Wave) 및 다중 모드(Multimode) 중첩을 실시간으로 시각화하는 웹 시뮬레이터의 개발 및 물리 모델 정리 문서입니다.1. 📌 개발 목표 및 개요목적: FDTD(Finite-Difference Time-Domain) 방식의 높은 연산 과부하를 극복하고, 웹 브라우저 환경에서 60 FPS로 작동하는 반-해석적(Semi-Analytical) 2D/3D 연동 도파로 시뮬레이터 구축.주요 시각화 요소:3D 공간: 외부 입사 레이저 빔(Cylinder Mesh) 및 Pitch/Yaw 각도, 3D 도파로 구조물.우측 하단 2D Canvas: 도파로 출구 단면($z = L$)에서의 전기장 $\vert{}E(x, y)\vert{}$ 히트맵 및 모드 제어.우측 상단 Canvas: 입력 레이저 파장 및 펄스 폭 스펙트럼(Gaussian Spectrum) 그래프.2. 🔬 핵심 물리 및 수학적 모델① 유효 굴절률 근사 모델 (Effective Index Approximation)3차원 직사각형 도파로($450 \times 220\text{ nm}$)의 기하학적 구속 효과(Confinement Factor)를 반영하기 위해 1차 수직 결합 근사식을 적용합니다.$$n_{\text{eff}} = \sqrt{n_1^2 + \Gamma_y \cdot (n_2^2 - n_1^2)}$$$$\text{NA}_{\text{eff}} = \sqrt{n_{\text{eff}}^2 - n_1^2}$$Silicon Core ($n_2$): $3.45$$\text{SiO}_2$ Cladding ($n_1$): $1.45$표준 단일 모드 검증 ($450 \times 220\text{ nm}, \lambda = 1550\text{ nm}$):유효 굴절률 $n_{\text{eff}} \approx 2.45$ 반영 시 유효 $V$-number는 $V_x \approx 1.45 \le \pi/2$ 가 되어 완벽한 $\text{TE}_{00}$ 단일 모드로 작동합니다.② 규격화 주파수(V-number) 및 다중 모드(TE_mn) 계산코어 너비($w$)나 높이($h$)가 확장되거나 파장($\lambda_0$)이 줄어들면 $V$-parameter가 상승하여 고차 모드가 허용됩니다.$$V_x = \frac{\pi \cdot w}{\lambda_0} \text{NA}_{\text{eff}}, \quad V_y = \frac{\pi \cdot h}{\lambda_0} \text{NA}_{\text{eff}}$$$$m_{\text{max}} = \left\lfloor \frac{2 V_x}{\pi} \right\rfloor, \quad n_{\text{max}} = \left\lfloor \frac{2 V_y}{\pi} \right\rfloor$$우수 모드 (Even Mode: $m, n = 0, 2, 4 \dots$): $\cos(k_x x)$ 기반 대칭 파동기수 모드 (Odd Mode: $m, n = 1, 3, 5 \dots$): $\sin(k_x x)$ 기반 반대칭 파동기수 모드 부호 대칭성 보정: $x < 0$ 또는 $y < 0$ 영역에서 경계값 $E_b$의 부호를 유지하여 클래딩 경계면에서의 불연속 절단선(Discontinuity Line) 방지.③ 연속 경계 조건 (C0/C1 Continuity) & 2D 에바네센트 결합코어 경계면($\vert{}x\vert{} = w/2, \vert{}y\vert{} = h/2$)에서 접선 전계가 연속성을 가지며 지수적으로 감쇠하도록 연산합니다.에바네센트 침투 깊이 ($d_p$):$$d_p = \frac{\lambda_0}{2\pi \sqrt{n_{\text{eff}}^2 \sin^2\theta - n_1^2}}, \quad \alpha = \frac{1}{d_p}$$모서리(Corner) 영역 ($\vert{}x\vert{} > w/2, \vert{}y\vert{} > h/2$):격자형 갈라짐을 방지하기 위해 2D 유클리드 거리를 사용한 반경 방향 감쇠 적용:$$r = \sqrt{(\vert{}x\vert{} - w/2)^2 + (\vert{}y\vert{} - h/2)^2}$$$$E_{\text{corner}}(x,y) = (E_{bx} \cdot E_{by}) \cdot e^{-\alpha r}$$④ 전반사 차단 (Radiation Loss) 조건입사 레이저 각도($\theta_{\text{inc}}$)가 수용각(Acceptance Angle)을 초과할 경우:$$\theta_{\text{inc}} > \theta_{\text{acc}} = \arcsin\left(\frac{\text{NA}_{\text{eff}}}{n_1}\right)$$도파 조건이 파괴되며 2D 히트맵의 전계 세기가 즉시 $0$이 되고 화면에 RADIATION LOSS (No TIR) 경고 문구가 표시됩니다.
